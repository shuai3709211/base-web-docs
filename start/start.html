开始 ​

项目启动 ​

  • 前期准备

    • 请使用pnpm作为包管理工具，；

    node版本在18以上，如果本地还有其他项目使用低版本node请安装node版本管理工具， n、nvm 等；

    • 下载依赖的时候有时候可能会下载不下来，可以选择使用nrm包管理工具；
    • 尽量不要使用vscode内置的shell工具下载依赖，有些时候经常下载不下来，不清楚为什么；

  • nrm私仓切换源

  #安装 nrm
  npm install -g nrm

  #新增私仓地址   ws可以使用其他名字，只是一个关键字，后面切换的时候需要使用
  nrm add  ws http://172.16.125.240:1937/  
  
  #切换到私仓    如果需要还原到npm源   使用 nrm use npm 即可还原回去
  nrm use  ws
  
  #查看是否成功切换  如果ws那行前面有个*号证明切换成功 
  nrm ls
  • 运行项目
  #下载依赖  如果下载依赖有问题或者依赖版本下载不正确，请使用  pnpm store prune  命令清除下pnpm的缓存
  pnpm install   
  
  
  #运行
  npm run dev
  
  #格式化代码 + 校验代码格式
  npm run lint
  
  #打包
  npm run build

开发注意 ​

  • 大驼峰 = 首字母大写、分割大写
  • 小驼峰 = 首字母小写、分割大写

文件命名规则

  • 页面、组件命名规则"大驼峰"命名；
  • 文件夹、ts等文件命名命名"小驼峰"；

代码命名规则

  • ts： type、interface、class声明方式为""；
  • css： css命名方式用"-"分割；
  • html：标签写法（el-input、gs-spin）、props写法也是用"-"方式,比如说定义的名字为isShow，组件使用的时候应该是is-show而不是isShow；

网络请求相关

  • 所有请求都封装在api这个文件夹下面 文件命名规则为 "模块名Apis"；
  • get请求参数放到params里面；
  • post请求参数默认放到data里面，注意有些特别的post请求data跟params都需要；
