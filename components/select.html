Item 输入组件 ​

由输入框、选择器、单选框、多选框等控件组成

基础用法 ​

html

<template>
  <gs-select v-model="value"
             :options="options"
             :size="size"
             :disabled="disabled"
             :allow-create="allowCreate"
             :dict-key="dictKey"
             :value-key="valueKey"
             :clearable="clearable"
             :multiple="multiple"
             :collapse-tags="collapseTags"
             :collapse-tags-tooltip="collapseTagsTooltip"
             :multiple-limit="multipleLimit"
             placeholder="placeholder"
             @change="change" />
</template>

<script lang="ts">  // 这里缺少个setup

  const value = '';
  const options = [{value:'1',name:'选项一'}];
</script>

:::

参数说明类型可选值默认值

size控件大小stringmedium / small / mini—
multiple是否多选booleanfalse
dictKey字典key，字典管理中配置的key，无需其他操作string-
value-keyoptions中不是正常返回{value:'1',name:'选项一'}这种格式的，需要手工指定怎样取值object
